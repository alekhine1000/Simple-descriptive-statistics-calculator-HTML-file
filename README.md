# Simple-descriptive-statistics-calculator-HTML-file

A basic web-based tool, suitable for educators to teach studntes how to analyze CSV data with interactive histograms and box plots.

## Features
- Upload CSV files
- Interactive statistical analysis
- Histograms with normal distribution overlay
- Box plots with outlier detection
- Responsive design

## Usage
1. Upload a CSV file
2. Select a numeric column to analyze
3. View statistics, histogram, and box plotA simple statistics calculator with CSV upload, that plots  histograms,  box plots and  identifies outliers
